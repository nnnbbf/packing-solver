# Packing Solver

1. 关于 AI 工具使用
	- 使用的 AI 工具：ChatGPT / GitHub Copilot（代码提示与补全）/ Deespeek
	- AI 主要帮助：思路拆解、代码生成、BUG 定位与修复（如贴墙对齐、SAT 判定调整、邻接放置算法实现）、编写调试说明与变更记录。
	- 我自己完成并调整的关键逻辑：
	  - 将物体角度直接与真实墙边角度对齐（去除四舍五入到 90° 的逻辑），保证与斜墙紧贴。
	  - 修改 SAT 判定以允许边界相切（减少不必要间隙）。
	  - 实现并调优 `tryPlaceAdjacentToPlaced`，枚举已放置物的边并沿边滑动以尝试紧贴放置。
	  - 调整贴墙与网格搜索的步长策略以在精度和性能之间平衡。

2. 快速使用
	make

    example系列为输入 ans系列为解答
