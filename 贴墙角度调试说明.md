# 贴墙角度调试说明

本调试文档用于验证“物体边与墙边严格平行或垂直”放置逻辑。

## 1. 角度计算逻辑

- 设墙边与 X 轴夹角为 `wallAngle`。
- 物体可选角度：
  - `angle1 = wallAngle`      // 物体边与墙边平行
  - `angle2 = wallAngle + 90` // 物体边与墙边垂直
- 不再进行 roundToNearest90 操作，直接采用墙边实际角度。

## 2. 调试建议

1. **输出每次尝试放置的角度**
   - 在贴墙放置（tryPlaceAgainstWall）时，打印 angle1/angle2。
2. **可视化验证**
   - 输出物体四个顶点坐标，检查是否与墙边重合。
3. **特殊情况测试**
   - 斜墙（如 30°、45°、60°）场景，确保物体边能严格贴合。
   - 矩形房间，物体应为 0°/90°。
4. **误差检查**
   - 计算物体边与墙边的夹角，理论应为 0° 或 90°，允许极小浮点误差。

## 3. 示例调试输出

```
尝试贴墙放置: 墙边角度=33.7°
  angle1=33.7°  angle2=123.7°
  物体顶点: (x1,y1) (x2,y2) (x3,y3) (x4,y4)
  物体边与墙边夹角: 0.02°
```

## 4. 常见问题排查

- 若发现物体与墙边有缝隙或重叠，检查角度赋值是否正确。
- 若物体未能紧贴斜墙，排查坐标变换和旋转实现。
- 若仍有误差，可输出详细浮点数值进行比对。
